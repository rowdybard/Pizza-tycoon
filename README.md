# Pizza Tycoon

## Game Features
- Manage your own pizza restaurant.
- Create and customize your pizza recipes.
- Hire staff and manage their tasks.
- Compete in pizza-making contests.
- Expand your restaurant and unlock new locations.
- Marketing campaigns to attract more customers.

## Deployment Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/rowdybard/pizza-tycoon.git
   ```
2. Navigate to the project directory:
   ```bash
   cd pizza-tycoon
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the game:
   ```bash
   npm start
   ```
5. Open your browser and go to `http://localhost:3000` to start playing!

Enjoy your pizza business!
